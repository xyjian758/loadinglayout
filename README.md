# loadinglayout\r\n
不同状态显示不同页面的自定义布局\r\n
Step 1. Add the JitPack repository to your build file \r\n
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}\r\n
  
  Step 2. Add the dependency\r\n
  	dependencies {
	        implementation 'com.github.xyjian758:loadinglayout:0.0.1'
	}
